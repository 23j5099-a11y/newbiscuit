# NewBiscuit

SPEC.mdをもとにした、Nodeとブラウザで動くビスケット風の幼児向けビジュアルプログラミング環境です。

## 起動

```bash
npm start
```

ブラウザで http://localhost:3000 を開きます。

## できること

- 部品置き場の絵をステージへドラッグして置く
- 部品置き場の絵をメガネの「まえ」「あと」へドラッグしてルールを作る
- ステージ上とメガネ内の絵をドラッグして位置を変える
- メガネの左側と同じ配置がステージで見つかると、右側の配置へ繰り返し変化する
- 同時に合うメガネや配置が複数ある場合はランダムに1つが動く
- 作品をJSONとしてサーバー側に保存・読み込みする

保存データは `data/project.json` に作られます。このファイルは作品ごとの実データなので、初期状態ではGit管理から外しています。
